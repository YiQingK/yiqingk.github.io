Exploring Music Taste link to project website

Data Visualization Project at USF

This was a solo project done during my Data Visualization class at USF.

Background

This project explores music trends from Spotify's weekly charts and visualizing the data in different methods such as lollipop and bump charts. It also investigates music taste ona global scale through a map displaying top songs of available countries.

Accomplishments

  • Designed website from scratch using HTML, CSS
  • Used D3.js to generate charts(line, bump, lollipop & map) along with tooltips and sliders.
  • Received feedback from peers through user testing and implemented new features accordingly
