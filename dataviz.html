Data Visualization Project at USF

This was a solo project done during my Data Visualization class at USF.

Project length: ~2 months

Goal: To represent a set of data through different charts using D3.js

Background

This project explores music trends from Spotify's weekly charts and visualizing the data in different methods such as lollipop and bump charts. It also investigates music taste ona global scale through a map displaying top songs of available countries.

Accomplishments

  • Coded website from scratch using HTML, CSS and Javascript
  • Used D3.js library to generate charts(line, bump, lollipop & map) along with tooltips and sliders.
  • Received feedback from peers through user testing and implemented new features and fixed problematic areas accordingly

Challenges Faced

After completing the beta version, I had received feedback from my peers which included points such as

  • Illegible chart legends for the line chart
  • Line chart is misleading as data starts and ends at different points
  • Chart and tooltip misalignment between browsers

Further work...

With more time, I would like to add a slider or timescale for the map, to be able to view data from different weeks as well as to add more bump charts with an accompanying selector in the form of checkboxes to toggle display on and off. Finally, I would like to be able to solve the issue of the wrapping of yaxis tick labels and its animation where they currently are unable to both be on.
