Work Scheduling App

Capstone Project at USF

This was a capstone project done in my final semester at USF sponsored by SF Dev Shop.

Project length: ~3 months

Team members: 4

Background

The App was created with the intention to provide an improved form of our sponsor's current system. We created an easy to use cross-platform mobile application to help organise a company's work schedule. The app was created using React Native for a cross-platform UI and it communicates with a back-end service built with Node.js and PostgreSQL.

Responsbilities/Accomplishments

  • 1 of 2 team members to be in-charge of the frontend development
  • Implemented stripe to the application for monetary processing of subscription, both frontend and backend
  • Designed a clock-in clock-out system for employees
  • Created user personas of targeted users as a team
  • Created workflow diagrams as well as low and high fidelity prototypes
